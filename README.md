# University Pathway Navigation System

## Overview
The University Pathway Navigation System is a C++ application developed as part of my MCA coursework. It demonstrates the use of graph data structures and graph algorithms to represent a university campus and determine efficient routes between different locations.

The project is intended for learning and understanding concepts such as graphs, shortest path algorithms, and data structures in C++.

## Features
- Represent campus locations as graph vertices.
- Connect locations using weighted edges.
- Find the shortest path between two locations.
- Display campus locations and connections.
- Console-based interface.
- Object-oriented implementation in C++.

## Technologies Used
- C++
- Standard Template Library (STL)
- Graph Data Structure
- Priority Queue
- Dijkstra's Algorithm
